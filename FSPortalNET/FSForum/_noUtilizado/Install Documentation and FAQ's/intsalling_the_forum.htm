Installing Web Wiz Forums
Return to the main menu

  1. Check with your web host that your web space is running on Windows NT4, 2000, 2003 operating system, with IIS 4+ and is ASP enabled. Web Wiz Forums will NOT run on a Unix/Linux web server! Please read 'Forum Requirements' for a full list of server requirements for this software.
  2. Check with your web hosts that you can use a DSN-less session("conn")ection to an Access database on your web space and that you have sufficient permissions to write to this database. If your hosts require that you have the forum's Access database in a specially set up directory please see Moving the Location and/or Renaming the Forum Database.
  3. Unzip the Forum zip file onto your hard drive keeping the files and images in there original directories.
  4. Upload the files and images through FTP to your webspace, again keeping the files and images in their directories. With ASP you don't need to place the files in a CGI bin.
  5. Once all the files are uploaded point your web browser to the location on your web space where you uploaded the forum to.
  6. Now that your forum is uploaded and running see, Setting up and Administering the Discussion Forum, to find out how to set up forums and administer your forum.
  7. If you find that you are getting errors and pages not displaying please see Troubleshooting and Common Errors.
  8. To login into the forum the default admin usuario and clave is: -
    • usuario: - Administrator
    • clave: - letmein

     

WARNING PLEASE NOTE:- For extra security from hackers make sure you don't leave your database unsecured, follow the instructions on the following page to secure your database now!!! - Moving the Location and Renaming the Forum Database
