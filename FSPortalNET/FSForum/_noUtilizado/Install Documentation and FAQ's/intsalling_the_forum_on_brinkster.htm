Installing Web Wiz Forums on Brinkster (free accounts)
Return to the main menu

  1. Unzip the Forum zip file onto your hard drive keeping the files in their original directories.
  2. Edit the two files called common.aspx with a text editor like notepad. One common.aspx file is with the main files and the other should be in a directory called admin.
    1. In both these files you should find the following line: -

      'strDbPathAndName = Server.MapPath(&quot;/usuario/db/wwForum.mdb&quot;)

    2. Remove the single quote mark ' shown in red above from the beginning of this line.
    3. Place your Brinkster usuario in the place, usuario, marked above in red.
  3. Edit the common.aspx file found in the main forum folder and change the following variable: -

    blnUseApplicationVariables = true

    To: -

    blnUseApplicationVariables = false

  4. Save the amended files.
  5. Upload the database called wwForum.mdb found in the admin/database directory of the forum to the directory called, db, on your Brinkster web space.
  6. Create a new folder on your Brinkster web space called forum.
  7. Create the following directories in the forum directory you have just created.
    1. admin
      1. database
      2. functions
      3. images
      4. includes
      5. security_images
    2. avatars
    3. forum_images
    4. functions
    5. includes
    6. language_files
    7. security_images
    8. smileys
    uploads
  8. Now upload all the Web Wiz Forums files and images to there respective folders you have created on your Brinkster web space.
  9. Once all the files are uploaded type the web address (URL) in your web browser to the location on your Brinkster web space where you uploaded the forum to so you can view the forum in your browser.
  10. If you have any problems trying to run the forum check that you have carried out all the above steps and edited both common.aspx files correctly.
  11. If the forum still isn't running try signing up to Brinkster again as it is not unheard of for Brinkster accounts to not be set up correctly.
  12. To login into the forum the default admin usuario and clave is: -
    • usuario: - Administrator
    • clave: - letmein

Important Note For Free Hosting Accounts:-

Due to the way free web hosts setup sites some features of Web Wiz Forums can not be used and will need to be disabled from the admin area once Web Wiz Forums is setup including:-

  1. Make sure you have followed the instructions above and have disabled the 'blnUseApplicationVariables', found in the common.aspx file otherwise your forum will NOT run!!!
  2. WYSIWYG (RTE) Post Editor - The adds that free web hosts inject into pages prevent this feature from running
  3. All email features - Free web hosts don't generally have email components installed and/or disabled with their free accounts
  4. Active Users List - Your site will be sharing server resources with other sites this prevents this feature from running correctly
  5. If you are using a free account from 1asphost, domaindlx, or Brinkster then the buttons for the post editor will not work as their add '100mb webspace etc.', is injected into a JavaScript file making the JavaScript within the file fail to run.
