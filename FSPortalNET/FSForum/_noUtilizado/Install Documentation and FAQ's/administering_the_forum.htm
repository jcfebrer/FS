Administering the Web Wiz Forums
Return to the main menu

  1. Once the your Web Wiz Forums is up and running you can administer and set up your forum by logging in through the forums login page with the default admin usuario and clave is: -
    • usuario: - Administrator
    • clave: - letmein
  2. Now you have logged into the forums admin account click on the admin link in the top left corner of the forum to take you to the administration and set up pages.
  3. In the admin section of the forum you can find the following:-
    • Set up new forums and categories - including security settings
    • Change the default admin usuario and clave - highly recommend
    • Setup and turn on and off email functions of the forum
    • Configure the forum - turn on and off forum functions and change the forum logo
    • Set up and edit User Groups - including ladder Groups
    • Set Group Permissions
    • Set individual User Permissions
    • Set up moderators and moderator groups
    • Suspend new user registration
    • Configure time and date settings
    • Configure file, image, and avatar uploading
    • Mass Email Members
    • Anti-spam configuration
    • Configure the Bad Word Filter
    • Forum Statistics
    • MS SQL Server Statistics (MS SQL Server Version)
    • Compact and Repair Database (Access Version)
    • IP Address Banning
    • Email Address Banning
    • Lock Forums
    • Re-sync Topic and Post Count
    • Batch Lock Old Topics
    • Batch Delete Topics
    • Batch Move Topics
    • Batch Delete Old Private Messages
    • Batch Delete Members
