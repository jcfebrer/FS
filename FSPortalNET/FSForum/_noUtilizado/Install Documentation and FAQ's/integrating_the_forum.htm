Integrating the Forum Into Your Web Site
Return to the main menu

As well as changing the look and feel of the forum through the skin file you can also easily integrate the forum into your own site by editing two files to add your own sites logo across the top, and any info you have in the footer of your site. The following needs to be done locally on your own machine then uploaded to your web space.

  1. Make a back-up of the files, header.aspx, and footer.aspx found in the includes directory.
  2. Open the file header.aspx in an HTML editor, locally on your own machine, (it is strongly recommended that you do NOT use Frontpage) and add your own web sites logo or any other content you wish display at the top of each page of the forum.
  3. Once you have finished amending the header.aspx file save the amended version, upload to your web space, and test the forum in your web browser to see if it looks and runs correctly.
  4. Open the file footer.aspx in an HTML editor, locally on your own machine, and add any content that you would like displayed at the bottom of each page of the forum.
  5. Once you have finished amending the footer.aspx file save the amended version, upload to your web space, and test the forum in your web browser to see if it looks and runs correctly.
  6. For more advanced layouts, such as a side bar navigation, you need to start the table containing the side bar navigation in the header.aspx file and close the table in the footer.aspx file.
